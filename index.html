Конвертер валют

Написано при помощи библиотек requests и bs4.

Конвертер валют с графическим интерфейсом

Динамика акций в реальном времени

Написано при помощи библиотек requests и bs4.

Программа делает запрос на указанные сайты с акциями, выбирает нужные данные и их выводит в консоль с задержкой в 2 секунды.

В репозитории лежит GUI версия 0.1 с графическим интерфейсом. Интерфейс в стадии разработки.

Запись данных в Excel-файл

Более продвинутая версия программы "Promotions Dynamics", написано при помощи библиотеки Pandas.

Программа получает актуальные на текущий момент данные об акциях и записывает их в Excel-файл.


Каждый файл пронумерован и датирован автоматически.
